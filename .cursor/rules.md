# Project Context

This is a portfolio website built with the following stack:

## Framework

- **Next.js** - React framework for production

## Deployment

- **Vercel** - Deployed and hosted on Vercel platform

## Content Management

- **Contentlayer** - Content SDK that validates and transforms content
- **Notion** - Used as a CMS together with Contentlayer for managing portfolio content

## Package Manager

- Uses **pnpm** for package management
